Also to control technicians, maintenance types, and customers related to devices. This project is a practical example for the last section of my course "Laravel From Scratch". Dynamic events calendar, Participant registration statistics, and Dynamic video gallery

  • Calendar to display events already stored in the database, with a second view to edit information related to each event. In this development, I used the FullCalendar library, Laravel, Javascript, Bootstrap, and SQL server as a database manager.
  • I developed a view in Laravel to display the graphs of various statistics of the participants registered in an event. I also used JavaScript, Bootstrap, and SQL Server.
  • I developed a small module to manage a dynamic video gallery. Once the information is configured and saved in the database (SQL Server), they are shown in this view to be displayed, you can also use the search field and the "Category" button to filter them. PHP, vanilla CSS, and JavaScript were used.

These developments described above were successfully integrated into a larger system that serves to manage face-to-face, virtual, or hybrid events.
